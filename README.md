A simple Address Book Management System built to store and manage contact information efficiently. This project allows users to add, view, search, update, and delete contacts using basic programming concepts and file handling.
The application demonstrates the use of structures, arrays/linked lists, functions, and CRUD operations, making it ideal for beginners and students learning data structures and system programming.

Key Features:
Add new contacts (Name, Phone, Email, Address)
View all saved contacts
Search contacts by name or phone number
Update existing contact details
Delete contacts
Persistent storage using files

Tech Stack:
Language: C
Concepts: Structures, File Handling, Functions, Loops
This project is suitable for academic projects, practice, and understanding real-world data management applications.
